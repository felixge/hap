Contributing
=======================
All attempts to improve this software are greatly appreciated.

Discussions
-----------
Open an issue.

Standards
---------
Follow the rules of `gofmt`. A pre-commit hook is highly recommended.

Develop
-------------
1. Fork.
2. Write tests.
3. Write code.
4. Submit Pull Requests.
5. Repeat 2-4
